PDF Tools

所有文件在浏览器本地处理

无需上传服务器，安全私密

合并 PDF

将多个PDF文件合并为一个文件。拖拽调整顺序后点击合并。

拖拽PDF文件到此处 或 点击选择文件

支持同时选择多个文件

拆分 PDF

提取指定页面或将每一页拆分为独立的PDF文件。

拖拽PDF文件到此处 或 点击选择文件

📄 提取指定页面

输入页码范围，例如: 1,3,5-8

📑 拆分所有页面

将每一页导出为单独的PDF文件并打包下载

删除页面

从PDF中删除指定的页面。

拖拽PDF文件到此处 或 点击选择文件

要删除的页面

输入要删除的页码范围，例如: 1,3,5-8

旋转 PDF

旋转PDF中的全部或部分页面。

拖拽PDF文件到此处 或 点击选择文件

旋转角度

指定页面范围（留空则旋转所有页面）

重排页面

上传PDF后可拖拽缩略图调整页面顺序，也支持反转和自定义顺序。

拖拽PDF文件到此处 或 点击选择文件

💡 拖拽缩略图即可调整页面顺序 · 新顺序号会实时更新

编辑 PDF

在PDF页面上添加文字和图片标注，然后导出为新PDF。

拖拽PDF文件到此处 或 点击选择文件

150% 点击标注选中 · 拖拽移动 · 四角缩放 · Del删除 · 双击文字编辑

翻译 PDF

选中文字即时翻译，或全文翻译中英对照阅读。

拖拽PDF文件到此处 或 点击选择文件

→
鼠标选中PDF文字即可看到翻译
点击「开始全文翻译」查看对照结果

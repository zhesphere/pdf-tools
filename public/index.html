PDF Tools

所有文件在浏览器本地处理

无需上传服务器，安全私密

合并 PDF

将多个PDF文件合并为一个文件。拖拽调整顺序后点击合并。

拖拽PDF文件到此处 或 点击选择文件

支持同时选择多个文件

拆分 PDF

提取指定页面或将每一页拆分为独立的PDF文件。

拖拽PDF文件到此处 或 点击选择文件

📄 提取指定页面

输入页码范围，例如: 1,3,5-8

📑 拆分所有页面

将每一页导出为单独的PDF文件并打包下载

删除页面

从PDF中删除指定的页面。

拖拽PDF文件到此处 或 点击选择文件

要删除的页面

输入要删除的页码范围，例如: 1,3,5-8

旋转 PDF

旋转PDF中的全部或部分页面。

拖拽PDF文件到此处 或 点击选择文件

旋转角度

指定页面范围（留空则旋转所有页面）

重排页面

反转页面顺序或自定义新顺序。

拖拽PDF文件到此处 或 点击选择文件

🔄 反转页面顺序

完全反转PDF页面的排列顺序（最后一页变第一页）

🔢 自定义顺序

输入新的页面排列顺序(逗号分隔)，例如: 3,1,2

PDF 信息

查看PDF文件的详细信息，包括页数、页面尺寸、元数据等。

拖拽PDF文件到此处 或 点击选择文件
